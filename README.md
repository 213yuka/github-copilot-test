# github-copilot-test

## 日本語To Doアプリ

Pythonで作成したシンプルなコマンドライン版のTo Doアプリケーションです。

### 機能

- ✅ タスクの追加
- ✅ タスク一覧の表示
- ✅ タスクの完了
- ✅ タスクの削除
- ✅ タスクの永続化（JSONファイルに保存）

### 必要要件

- Python 3.6以上

### 使い方

1. アプリを起動:

```bash
python todo_app.py
```

2. メニューから操作を選択:
   - `1`: 新しいタスクを追加
   - `2`: すべてのタスクを表示
   - `3`: タスクを完了にする
   - `4`: タスクを削除
   - `5`: アプリを終了

### 使用例

```
To Doアプリへようこそ！

=== To Doアプリ ===
1. タスクを追加
2. タスク一覧を表示
3. タスクを完了
4. タスクを削除
5. 終了

選択してください (1-5): 1
タスクの内容: 買い物に行く
✓ タスクを追加しました: 買い物に行く
```

### データ保存

タスクは `tasks.json` ファイルに自動的に保存されます。
